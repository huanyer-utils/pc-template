# 项目说明
### 依赖安装

由于使用了私有npm仓库，所以请使用私有源安装依赖
```
npm install --registry=http://223.4.78.73:4873
```

vite方式启动
```
npm run dev
```

### 构建方式

- 查看打包大小的报告

```
npm run build:report
```

- 测试环境打包

```
npm run build:dev
```

- 线上环境打包

```
npm run build
```

- 输入 basename，并选择打包环境

```
npm run build:basename
```
